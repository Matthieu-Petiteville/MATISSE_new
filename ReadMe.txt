Step 1 :
Set an environment variable : MATISSE_HOME as your local folder for MATISSE
Set an environment variable : MATISSE_DATA as your local folder for MATISSE's data
Set an environment variable : R_HOME as your R install folder (for instance : F:\R-4.0.4')
How to set environment variable (Win 10) : Search 'variable env' select 'Modify environment variable'.
Select 'Env variable', then 'New' in 'System variable'. Name = MATISSE_HOME, Variable value = Folder where MATISSE is installed
