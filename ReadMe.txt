Step 1 :
Set an environment variable : MATISSE_HOME as your local folder for MATISSE
Set an environment variable : MATISSE_DATA as your local folder for MATISSE's data
How to set environment variable (Win 10) : Search 'variable env' select 'Modify environment variable'.
Select 'Env variable', then 'New' in 'System variable'. Name = MATISSE_HOME, Variable value = Folder where MATISSE
is installed

Step 2 :
In MATISSE_DATA, create the folder :
MATISSE_DATA/Output/Step_O - folder for all the standard data that will be used during MATISSE run
